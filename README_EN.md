# Standard Solution Review System

![Python](https://img.shields.io/badge/Python-3.12-blue)
![PyQt6](https://img.shields.io/badge/PyQt6-GUI-green)
![SQLite](https://img.shields.io/badge/SQLite-Database-blue)
![GPL-3.0](https://img.shields.io/badge/GPL--3.0-red)

A desktop application designed for the standardization, calculation, and review of laboratory volumetric standard solutions.

The software supports automatic temperature correction, burette calibration correction, four-replicate calculations, eight-replicate calculations, relative range evaluation, and final concentration reporting.

Simply enter the required data, and the software will automatically complete the calculation.

![](https://img.weigeshare.cc.cd/img/002.StandardSolution_Review_System_Demo_GIF.gif)

Simply enter the required data and calculate instantly.

## Features

✅ Temperature Correction

✅ Burette Calibration Correction

✅ Automatic Standard Solution Concentration Calculation

✅ Four-Replicate Calculation

✅ Eight-Replicate Calculation

✅ Relative Range Evaluation

✅ Automatic Reported Concentration Calculation

## Supported Standard Solutions

- Hydrochloric Acid, Sodium Hydroxide, Sulfuric Acid, Potassium Permanganate, Silver Nitrate, Sodium Thiosulfate, Ethylenediaminetetraacetic Acid (EDTA), Zinc Chloride, Potassium Hydroxide-Ethanol, Sodium Carbonate, etc.

- HCl, NaOH, H₂SO₄, KMnO₄, AgNO₃, Na₂S₂O₃, EDTA, ZnCl₂, KOH-Ethanol, Na₂CO₃

- **Customizable**

## Usage

### Release Version

Download and run the [EXE](https://github.com/weigefenxiang/StandardSolutionReviewSystem/releases) latest 

```text
StandardSolution_ReviewSystem.exe
```

## Run from Source

### Development Environment

- Windows 11
- Python 3.12
- PyQt6
- QFluentWidgets
- SQLite
- Nuitka (for packaging)

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python Flu_Main.py
```
