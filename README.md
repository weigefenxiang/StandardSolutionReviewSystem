## Introduction | 项目简介

**Standard Solution Review System** is a laboratory software developed for the preparation, standardization, review, and quality verification of volumetric standard solutions.

The software automates concentration calculations, temperature corrections, burette calibration corrections, relative range determination, and final concentration reporting, improving calculation efficiency and reducing human error.

**标准溶液计算审核系统** 是一款用于实验室标准溶液配制、标定、审核及质量控制的软件。

系统可自动完成：

- 温度校正
- 滴定管校正
- 浓度计算
- 单人四平行计算
- 双人八平行计算
- 相对极差计算
- 标液报出浓度计算

从而减少人工计算错误，提高实验室工作效率。
